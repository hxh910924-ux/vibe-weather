# UI 设计文档

> 项目：氛围天气
> 状态：✅ 已确认

---

## 一、整体风格

- **风格**：全屏沉浸式，无边框无卡片，信息悬浮于动效之上
- **主色调**：随天气动态变化，不设固定主色
- **字体**：系统默认无衬线字体，温度数字用超轻字重（font-weight: 200），营造通透感
- **文字颜色**：白色半透明白（rgba(255,255,255,0.9)），深色天气时可适当提高不透明度
- **圆角**：所有弹窗和按钮使用 16px 圆角
- **模糊**：大量使用 backdrop-filter: blur() 毛玻璃效果

## 二、页面结构

### 主页面（唯一页面）

**页面草图（手机竖屏）：**

```
┌─────────────────────────────────┐
│                                 │
│         ☁️ 动效背景层            │  ← Canvas 全屏动效
│         （雨滴/雪花/光线等）      │
│                                 │
│                                 │
│           北京市                  │  ← 城市名，小字，半透明，可点击
│                                 │
│           23°                   │  ← 温度，超大字号，轻字重
│         小雨 · 东南风3级          │  ← 天气概况，中等字号
│                                 │
│     🌅 06:12    🌇 18:35        │  ← 日出日落，小字，带图标
│                                 │
│                                 │
│    今天  明天  后天  周四  周五    │  ← 日期选择栏，半透明横条
│    ···   ···   ···   ···   ···  │     当前日期有微光高亮
│                                 │
│                                 │
│  ☀️ ⛅ 🌧 ❄️ 🌫 🌙             │  ← 底部主题切换，小圆点图标
│                                 │
└─────────────────────────────────┘
```

**各区域说明：**

| 区域 | 内容 | 说明 |
|------|------|------|
| 动效背景层 | Canvas 全屏渲染 | 占满整个视口，z-index: 0 |
| 城市名 | 当前城市名称 | 顶部偏上，小字号(14px)，半透明白色，cursor: pointer |
| 温度 | 数字 + ° | 页面中心偏上，字号 120px，font-weight: 200 |
| 天气概况 | 降水 + 风力 | 温度下方，字号 16px，rgba(255,255,255,0.7) |
| 日出日落 | 时间 + 图标 | 天气概况下方，字号 14px，间距 40px |
| 日期选择栏 | 7天日期横排 | 页面下方 1/4 处，flex 横排，间距 24px |
| 主题切换 | 6个天气图标 | 页面底部，小圆点(32px)，默认半透明，悬停显示标签 |

## 三、交互设计

### 城市搜索的交互

| 用户操作 | 界面反应 | 说明 |
|----------|----------|------|
| 点击城市名 | 弹出毛玻璃搜索弹窗 | 弹窗从顶部滑下，backdrop-filter: blur(20px) |
| 输入城市名 | 实时显示搜索结果列表 | 每条结果显示城市名+国家，点击即选 |
| 点击搜索结果 | 弹窗关闭，天气数据和动效过渡更新 | 过渡动画 1 秒 |
| 点击弹窗外部 | 弹窗关闭 | - |
| 首次打开（有历史城市） | 搜索框下方显示"最近"城市标签 | 最多 5 个，点击即选 |

**搜索弹窗草图：**

```
┌─────────────────────────────────┐
│  🔍 搜索城市...                  │  ← 输入框，毛玻璃背景
├─────────────────────────────────┤
│  最近：北京  上海  广州  深圳  成都  │  ← 历史城市标签
├─────────────────────────────────┤
│  北京市, 中国                     │  ← 搜索结果
│  京东, 中国                      │
│  京山, 中国                      │
└─────────────────────────────────┘
```

### 日期切换的交互

| 用户操作 | 界面反应 | 说明 |
|----------|----------|------|
| 点击某天日期 | 该日期高亮，天气数据和动效过渡 | 过渡动画 1 秒 |
| 点击今天 | 回到当天天气 | 同上 |

### 主题切换的交互

| 用户操作 | 界面反应 | 说明 |
|----------|----------|------|
| 悬停底部图标 | 图标放大到 40px，显示文字标签 | opacity 过渡 |
| 点击图标 | 动效立即切换到对应主题 | 1.5 秒淡入淡出 |
| 当前激活的主题 | 图标完全不透明 | 与其他半透明图标区分 |

### 加载状态

| 场景 | 界面反应 | 说明 |
|------|----------|------|
| 首次加载 | 显示"正在获取天气..."文字，半透明 | 数据到位后淡出 |
| 切换城市/日期 | 动效渐变过渡，数据淡入 | 不显示 loading 菊花 |
| API 失败 | 显示"天气数据暂时获取不到"文字，保持当前动效 | 融入背景风格 |

## 四、组件清单

| 组件 | 用途 | 备注 |
|------|------|------|
| 全屏 Canvas | 渲染天气动效（雨滴/雪花/光线/云层） | 占满视口，position: fixed |
| 城市名文本 | 显示当前城市，可点击 | cursor: pointer，hover 时下划线 |
| 温度显示 | 大字号温度数字 | font-weight: 200 |
| 天气概况文本 | 降水+风力信息 | rgba(255,255,255,0.7) |
| 日出日落行 | 图标+时间 | Flex 布局，间距 40px |
| 日期选择栏 | 7天横排选择器 | 当前日期高亮，半透明背景条 |
| 主题切换栏 | 6个天气图标 | 底部固定，小尺寸，半透明 |
| 城市搜索弹窗 | 毛玻璃搜索+结果列表 | backdrop-filter: blur(20px) |
| 加载/错误提示 | 文字提示 | 半透明白色，融入背景 |
